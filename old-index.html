The most exciting use of technology

Unique, non-standard practical JavaScript usage, that you'd never think of. Mix with non traditional software and technologies, that makes the language shine boosting development/maintenance qualifiers.

  • CodeSandbox

    codesandbox.io

    CodeSandbox is an online editor that helps you create web applications, from prototype to deployment

  • Serverless LibreOffice

    github.com/vladgolubev/serverless-libreoffice

    JavaScript driven serverless office suite

  • tui.editor

    github.com/nhnent/tui.editor

    Command line tool to optimize your webfont loading. Aggressive subsetting based on your font use, self-hosting of Google fonts and preloading

  • isomorphic-git

    github.com/isomorphic-git/isomorphic-git

    isomorphic-git is a pure JavaScript implementation of git that works in node and browser environments (including WebWorkers and ServiceWorkers)

Breakthrough of the year

Project contributing the most to the JavaScript ecosystem, adding new dimensions to it and possibilities for further development. New concepts/ideas with big future potential and good first realization in 2017.

  • TensorFlow.js

    js.tensorflow.org/

    A JavaScript library for training and deploying ML models in the browser and on Node.js

  • Hyperapp

    github.com/hyperapp/hyperapp

    Hyperapp is a JavaScript micro-framework for building web applications.

  • Apollo GraphQL

    www.apollographql.com

    Apollo provides a universal GraphQL API on top of your existing services, so you can build new application features fast without waiting on backend changes.

  • Fastify

    github.com/fastify/fastify

    Fastify is a web framework highly focused on providing the best developer experience with the least overhead and a powerful plugin architecture.

  • sixdegreesofwikipedia
