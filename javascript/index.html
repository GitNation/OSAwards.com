2019 Nominees

The most exciting use of technology

Unique, non-standard practical JavaScript usage, that you'd never think of. Mix with non traditional software and technologies, that makes the language shine boosting development/maintenance qualifiers.

  • List

    github.com/funkia/list

    An immutable list with unmatched performance and a comprehensive functional API.

  • mermaid

    github.com/knsv/mermaid

    Generation of diagram and flowchart from text in a similar manner as markdown.

  • flexsearch

    github.com/nextapps-de/flexsearch/

    Next-Generation full text search library for Browser and Node.js.

  • picojs

    github.com/tehnokv/picojs

    A face detection library in 200 lines of JavaScript.

Breakthrough of the year

Project contributing the most to the JavaScript ecosystem, adding new dimensions to it and possibilities for further development. New concepts/ideas with big future potential and good first realization in 2017.

  • hybrids

    github.com/hybridsjs/hybrids/

    Web Components from plain objects and pure functions!

  • comlink

    github.com/GoogleChromeLabs/comlink

    Comlink makes WebWorkers enjoyable.

  • kepler.gl

    kepler.gl

    Kepler.gl is a data-agnostic, high-performance web-based application for visual exploration of large-scale geolocation data sets.

  • strapi

    github.com/strapi/strapi

    Open source Node.js Headless CMS to easily build customisable APIs.

Fun side project of the year

Fun projects that would make you go "wow", without specific purpose or logic, but interesting and entertaining.

  • signale

    github.com/klaussinani/signale

    Highly configurable logging utility.

  • wired-elements

    github.com/wiredjs/wired-elements

    Collection of custom elements that appear hand drawn. Great for wireframes or a fun look.

  • terminalizer

    github.com/faressoft/terminalizer

    Record your terminal and generate animated gif images or share a web player.

  • curtainsjs

    github.com/martinlaxenaire/curtainsjs

    curtains.js is a lightweight vanilla WebGL javascript library that turns HTML elements into interactive textured planes.

Most impactful contribution

Most notable impact to the community development and growth.

    github.com/immerjs/immer

    Create the next immutable state by mutating the current one.

  • kutt

    github.com/thedevs-network/kutt

    Free Modern URL Shortener.

  • dayjs

    github.com/iamkun/dayjs

    Day.js 2KB immutable date library alternative to Moment.js with the same modern API.

  • apollo-server

    github.com/apollographql/apollo-server

    GraphQL server for Express, Connect, Hapi, Koa and more.

Productivity booster

The best to improve development efficiency.
